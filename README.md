# About
UKernel is a small kernel made in C and ASM.
# Building

## Prerequisites
**NASM**
**GCC**
**Make**
*Optional (for testing):* **QEMU**

If you're running Ubuntu, you can simply run:
`sudo apt install build-essential nasm qemu`

## Building and running
Build: `make`
Run: `make test`
Clean: `make clean`
